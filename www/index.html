Formatting

Formatting Numbers

Rounding a number Round ([AREA], 1) "12374568.2"

Formatting currency [COST] = "1234567" FormatCurrency ([COST]) "1,234,567.00"

FormatNumber([POP_98], 2) Where 2 is the number of decimal places Convert square feet to acresround to 2 Digits and append "Acres": Round(([Area] * 0.00002295684), 2) & " Acres" "45.26 Acres"

Calculate a Percentage FormatNumber([RENTER_OCC] / [HOUSEHOLDS] ) * 100 & "%" "27%"

Formatting Text

"LOTPLAN" IN ( '21RP741159' , '51SP107866' ) returns rows with values that equal 21RP741159 and 51SP107866
NOT "LOTPLAN" IN ( '21RP741159' , '51SP107866' ) returns rows with values that are not 21RP741159 and 51SP107866
"LOTPLAN" NOT IN ( '21RP741159' , '51SP107866' ) returns rows with values that are not 21RP741159 and 51SP107866

SELECT

The SELECT * FROM clause is hard coded in ArcMap which restricts the SELECT clause to return all of the columns in the table. Opening the Select by Attribute Tool SELECT * FROM name_of_feature.shp (Select all columns from name_of_feature.shp table) This restricts the use of other SQL functionality like: DISTINCT, ORDER BY, GROUP BY. These however can be used in sub queries

HTML Decorations

Examples of HTML Syntax to decorate labels in ArcMap

Bold      maybe not so BEAUTIFUL

"<BOL>" & [FIELD] & "</BOL>"

Italic   Just like this

"<ITA>" & [FIELD] & "</ITA>"

Underline   downunder

"<UND>" & [FIELD] & "</UND>"

All capitals   London, Munich, New York

"<ACP>" & [FIELD] & "</ACP>"

Small capitals   Wagga Wagga, Timbuktu

"<SCP>" & [FIELD] & "</SCP>"

Superscript       Super5

"<SUP>" & [FIELD] & "</SUP>"

Subscript       Sub5

"<SUB>" & [FIELD] & "</SUB>"

Colour (RGB)

# Hint use single quotes around the RGB colour value
# You can use any value colour combination red, blue, green = {0-255}
"<CLR red = '255'>" & [FIELD] & "</CLR>"
"<CLR blue = '55'>" & [FIELD] & "</CLR>"
"<CLR green = '100'>" & [FIELD] & "</CLR>"

Colour (CMYK)

# Hint use single quotes around the CMYK colour value
# You can use any value colour combination cyan, magenta, yellow, black = {1-100}
"<CLR magenta = '100'>" & [FIELD] & "</CLR>"
"<CLR cyan = '50'>" & [FIELD] & "</CLR>"
"<CLR yellow = '75'>" & [FIELD] & "</CLR>"
"<CLR black = '80'>" & [FIELD] & "</CLR>"

Character Spacing        SubStretching

# Spacing is expressed as a percentage value to the default. 0% means no spacing adjustment.
# spacing = {1-}
<CHR> spacing = "300" Some Text <\CHR>

Character Width       Wide Load

# May not be noticable of some mobile devices. It works in ArcMap :-)
<CHR> width = "100" Wide load <\CHR>
